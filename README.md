## Promt de gemini ai
lenguaje Dart con el framework Flutter para un nivel de aprendizaje principiante-intermedio y consiste en una aplicación artística con navegación profesional basada en rutas nombradas desde el archivo principal. La primera pantalla funciona como un splash screen con fondo color crema donde se muestra de forma destacada el nombre de la alumna Melanie Garcia 6° I junto al logo de ArtStore y un indicador de carga circular, permaneciendo activa durante diez segundos antes de redirigir automáticamente a la segunda página. Esta segunda pantalla corresponde al inicio de sesión y mantiene la estética con un fondo claro, un título con icono de pincel y cuadros de texto para usuario y contraseña que utilizan un diseño ovalado tipo stadium con bordes café oscuro, incluyendo un botón artístico para acceder y otro para ir a la fase de registro. Finalmente, la tercera pantalla permite crear una cuenta manteniendo la armonía visual mediante campos de texto idénticos para nombre, apellido, correo y usuario, incorporando además un botón de retroceso funcional en la parte superior derecha con el icono de flecha para regresar al login, logrando así una interfaz atractiva, coherente y de fácil navegación para el usuario.
